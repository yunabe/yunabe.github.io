JVM (java, javac コマンド) の言語, ロケールを設定方法

質問

JVM (java, javac コマンド) の言語・ロケールはどうやって指定するの？

答え

user.language, user.country オプションを java や javac コマンドを実行する際に指定することでJVMのロケール（言語・国）のデフォルト値が設定できます。 user.language に指定できるのはISO 639 の2桁の小文字、 user.country に指定できるのはISO 3166 の2桁の大文字です。

java -Duser.language=ja -Duser.country=JP ...

環境変数 _JAVA_OPTIONS に設定しておけば、java, javac 起動時に自動的に引数が渡されます。

export _JAVA_OPTIONS="-Duser.language=ja -Duser.country=JP"
